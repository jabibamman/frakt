  • Result<TcpStream, std::io::Error> - This function returns a Result type. On success, it returns the modified TcpStream wrapped in Ok. On failure, it returns an std::io::Error.

This function prepares a message by calling prepare_message, clones the provided TCP stream, writes the prepared message to the cloned stream, and then returns the original stream.

use std::net::TcpStream;
use server::services::write::write;

let mut stream = TcpStream::connect("localhost:8787").unwrap();
let stream = write(&mut stream, "Hello world!").unwrap();
